UI Style Guides

Overview

Themes

There are a number of themes that you can use to make it easier to add data privacy oriented UI features to your flutter application. These themes include customizations and assets for all of the UI Components listed below and can be used application wide with your application or just selectively on specific components as you prefer.

List of Supported Themes

  • The @ Company theme
  • DarkMode
  • LightMode
